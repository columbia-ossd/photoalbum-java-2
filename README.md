# PhotoAlbum
This program allows users to list and view photos.

To run the program, run:
```
java App
```

# Configuration
The file "images.txt" contains the listing of photos and their info.

"Photo.java" and "Album.java" represent a photo and album class, respectively.

"App.java" contains the code to interact with the user.

"StdDraw.java" is a graphics library that should work on different platforms.
